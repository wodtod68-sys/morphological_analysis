# morphological_analysis
# Korean POS Corpus — Morpheme & Index Term Extractor

한국어 형태소 분석 결과(POS tagged corpus)로부터  
**(1) 형태소/품사를 추출**하고, **(2) 색인어(단일어·복합어)를 생성**하는 Python 스크립트입니다.

---

## 📁 파일 구성

```
.
├── get_morphs_tags.py   # 형태소/품사 추출
└── tagged2context.py    # 색인어 문맥 파일 생성 (get_morphs_tags.py 필요)
```

---

## ⚙️ 요구 사항

- Python 3.6 이상
- 외부 라이브러리 없음 (표준 라이브러리만 사용)

---

## 🚀 사용법

### get_morphs_tags.py

형태소 분석 파일을 받아 `형태소\t품사` 형식으로 출력합니다.

```bash
./get_morphs_tags.py <입력파일> > <출력파일>
```

```bash
# 예시
./get_morphs_tags.py all.tag > all.mt
```

### tagged2context.py

형태소 분석 파일을 받아 어절별 색인어를 추출하고 문맥 파일(`.context`)을 생성합니다.  
여러 파일을 한 번에 처리할 수 있습니다.

```bash
./tagged2context.py <입력파일(s)>
```

```bash
# 예시
./tagged2context.py all.tag          # all.tag.context 생성
./tagged2context.py *.tag            # 여러 파일 일괄 처리
```

> `tagged2context.py`는 `get_morphs_tags.py`를 모듈로 import하므로, **두 파일은 반드시 같은 디렉터리**에 있어야 합니다.

## 📄 입출력 형식

### 입력 파일

각 줄: `어절\t형태소분석결과` (탭 구분 2컬럼)  
빈 줄: 문장 경계

```
김현욱(金顯煜)사무총장도	김현욱/NNP+(/SS+金顯煜/SH+)/SS+사무총장/NNG+도/JX
최근	최근/NNG
합의정신에	합의정신/NNG+에/JKB

결국	결국/NNG
```

형태소 분석 결과의 기본 형식은 `형태소/품사+형태소/품사+...`입니다.

### 출력 — get_morphs_tags.py

```
김현욱	NNP
(	SS
金顯煜	SH
)	SS
사무총장	NNG
도	JX
최근	NNG
```

### 출력 — tagged2context.py (`.context`)

문장별로 한 줄, 색인어를 공백으로 구분합니다.

```
김현욱 金顯煜 사무총장 최근 공동정권 합의정신 수도권 양당 지분 ５대 기준 논의 강조 송인수 기자 송인수기자 
이만섭 대행 이만섭대행 자민련 비판 파장 2000년 새해 첫발 
```

## 🔍 구현 상세

### get_morphs_tags.py — 파싱 알고리즘

형태소 분석 결과는 단순 `+` split이나 마지막 `/` split으로는 처리할 수 없는 세 가지 특이 케이스가 있습니다.

| 케이스 | 예시 | 파싱 결과 |
|--------|------|-----------|
| 형태소 자체가 `+` | `BB/SL++/SW` | `('BB','SL'), ('+','SW')` |
| 형태소 자체가 `/` | `강원/NNP+//SP+인제/NNP` | `('강원','NNP'), ('/','SP'), ('인제','NNP')` |
| 형태소 내부에 `/` 포함 | `Supreme/Nike/NNP` | `('Supreme/Nike','NNP')` |

이를 해결하기 위해 **`/품사태그` 패턴을 직접 탐색**하는 방식을 사용합니다.  
품사 태그(영문 대문자) 직후가 `+`(구분자) 또는 문자열 끝일 때만 유닛 경계로 확정하고,  
그렇지 않으면 형태소 내부의 `/`로 판단해 탐색을 이어갑니다.

```
현재 위치 i에서 시작
  └─ '/' 탐색
      ├─ '/' 뒤가 영문 대문자(품사 태그)?
      │    ├─ 품사 태그 끝이 '+' 또는 문자열 끝?  →  ✅ 유닛 확정
      │    └─ 품사 태그 뒤에 다른 문자?           →  🔄 다음 '/'부터 재탐색
      └─ '/' 뒤가 품사 태그 아님?                 →  🔄 다음 '/'부터 재탐색
```

### tagged2context.py — 색인어 추출 규칙

#### 색인 대상 품사

| 품사 | 의미 | 단일어 색인 | 복합어 참여 |
|------|------|:-----------:|:-----------:|
| NNG | 일반명사 | ✅ | ✅ |
| NNP | 고유명사 | ✅ | ✅ |
| SH  | 한자 | ✅ | ✅ |
| SL  | 외국어(영어) | ✅ 단독일 때만 | ✅ |
| NR  | 수사 | ❌ | ✅ |
| NNB | 의존명사 | ❌ | ✅ |
| SN  | 숫자 | ❌ | ✅ |

> **SL 규칙**: 다른 색인 대상 형태소와 연속되어 복합어를 이루면 단일어로는 색인하지 않습니다.

#### 추출 알고리즘

**1단계 — 연속 그룹 분할**  
색인 대상 품사가 연속되는 구간을 하나의 그룹으로 묶습니다.  
비대상 품사(`JKB`, `SS`, `VV` 등)를 만나면 그룹이 끊깁니다.

```
Y/SL + 2/SN + K/SL + 위원회/NNG + 에/JKB
 ←────── 그룹 1 (4개 연속) ──────→  ↑끊김

휴렛팩커드/NNP + (/SS + HP/SL + )/SS + 사/NNG
     그룹1        끊김   그룹2   끊김   그룹3
```

**2단계 — 단일어·복합어 추출**  
출력 순서는 **단일어를 먼저, 복합어를 뒤에** 출력합니다.

```
# 단독 (그룹 크기 1)
KGB/SL          →  KGB
최근/NNG         →  최근

# 2개 연속
SK/SL + 텔레콤/NNG
  →  단일어: 텔레콤  (SK는 복합어 소속 SL → 단일어 제외)
  →  복합어: SK텔레콤
  →  출력:   텔레콤 SK텔레콤

# 3개 이상 연속 → 가장 긴 복합어 하나만 (중간 복합어 없음)
정부/NNG + 조직/NNG + 개편/NNG
  →  단일어: 정부 조직 개편
  →  복합어: 정부조직개편  ('정부조직', '조직개편' 등은 출력 안함)
  →  출력:   정부 조직 개편 정부조직개편

Y/SL + 2/SN + K/SL + 위원회/NNG
  →  단일어: 위원회  (Y·K는 복합어 소속 SL, 2는 SN → 모두 단일어 제외)
  →  복합어: Y2K위원회
  →  출력:   위원회 Y2K위원회

５/SN + 대/NNB
  →  단일어: 없음  (SN·NNB는 단일어 색인 안함)
  →  복합어: ５대
  →  출력:   ５대
```
